OpenClaw Control UI Overlay

Mission Control pages are mounted

This repository ships a UI overlay for an existing OpenClaw deployment. It wraps the native control UI, adds the Mission Control overview pages, and can optionally connect to a host-side helper for diagnostics.

What this package changes

Wraps: the native OpenClaw control UI entry at /
Adds: /mission-control.html and /mission-control-overview.html
Generates: a mountable custom-ui directory for /custom-ui
Does not replace: the OpenClaw backend or gateway
Optionally includes: a host-side helper for diagnostics and service visibility
